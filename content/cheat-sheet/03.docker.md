#### Docker Container

::md-code-block
```bash
docker ps -a
```
::

::md-description
List all containers
::

::u-separator{size="xs"}
::

::md-code-block
```bash
docker run -d --name <container_name> --network <network_name> --env-file <env_file> <image_name>
```
::

::md-description
Run new Container
::

::u-separator{size="xs"}
::

::md-code-block
```bash
docker stop <container_id>
```
::

::md-description
Stop running container
::

::u-separator{size="xs"}
::

::md-code-block
```bash

```
::

::md-description
