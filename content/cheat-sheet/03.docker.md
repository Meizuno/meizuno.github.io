### Docker Image

```bash [Terminal]
docker image ls -a
```

::description
List all images
::

---

```bash [Terminal]
docker build <context>
```

::description
Build image
::

---

```bash [Terminal]
docker rmi <image_id>
```

::description
Remove image
::

---

### Docker Container

```bash [Terminal]
docker ps -a
```

::description
List all containers
::

---

```bash [Terminal]
docker run -d --name <container_name> --network <network_name> --env-file <env_file> <image_name>
```

::description
Run new Container
::

---

```bash [Terminal]
docker stop <container_id>
```

::description
Stop running container
::

---

```bash [Terminal]
docker rm <container_id>
```

::description
Remove stopped container
::

---
